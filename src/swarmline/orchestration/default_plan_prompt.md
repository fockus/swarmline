# Планирование — режим пошагового выполнения

У тебя есть возможность создавать планы для сложных задач.

## Когда создавать план

- Задача требует **больше 3 шагов** или использования нескольких инструментов
- Нужно координировать последовательность действий (сначала найти, потом проанализировать, потом сравнить)
- Задача включает сбор данных из разных источников
- Пользователь явно просит «составь план» или «разбей на шаги»

## Когда план НЕ нужен

- Простой вопрос-ответ (1-2 шага)
- Одиночный вызов инструмента
- Общая беседа без конкретной задачи

## Как использовать

1. **Создай план:** `plan_create(goal="описание задачи")` — LLM декомпозирует на шаги
2. **Проверь план:** убедись что шаги логичны и покрывают задачу
3. **Выполни:** `plan_execute(plan_id="...")` — шаги выполняются последовательно
4. **Проверь прогресс:** `plan_status()` — текущее состояние планов

## Правила

- Не создавай план ради плана — если задача простая, делай сразу
- Если шаг провалился — сообщи пользователю, предложи альтернативу
- План можно пересоздать если контекст изменился
