<task id="2" name="Anthropic Adapter + LLM Client Wiring" owner="dev-infra" critical="yes" depends="1">
<action>
1. Extend `AnthropicAdapter.call()` in `runtime/thin/llm_providers.py`:
   - Pop `_thinking_config` from kwargs at start
   - When `_thinking_config` present: add `thinking={"type": "enabled", "budget_tokens": N}` to `messages.create()` kwargs
   - After response: iterate `response.content`, collect blocks with `hasattr(block, "thinking")` into thinking_parts
   - If thinking_parts non-empty: return `LlmCallResult(text=text, thinking="\n".join(thinking_parts))`
   - Otherwise return plain `str` as before

2. Extend `default_llm_call` in `runtime/thin/llm_client.py`:
   - When `config.thinking` is set and `resolved.provider == "anthropic"`:
     - Pop `stream` from kwargs (thinking requires full response)
     - Add `_thinking_config=config.thinking` to kwargs
   - After `adapter.call()`: handle `LlmCallResult` return type via isinstance check
   - Return `LlmCallResult` to caller

3. Extend `run_buffered_llm_call` in `runtime/thin/llm_client.py`:
   - After `llm_call()` returns: check `isinstance(result, LlmCallResult)` BEFORE `isinstance(result, str)`
   - If LlmCallResult: return `BufferedLlmAttempt(raw=result.text, chunks=[result.text], used_stream=False, thinking=result.thinking)`

4. Extend `try_stream_llm_call`:
   - After `llm_call()`: check `isinstance(result, LlmCallResult)` BEFORE `isinstance(result, str)`
   - If LlmCallResult: return `([result.text], result.text)` (thinking handled via BufferedLlmAttempt)

5. Emit thinking_delta in strategies:
   - `run_conversational` (conversational.py): after getting `attempt`, if `attempt.thinking` is set, yield `RuntimeEvent.thinking_delta(attempt.thinking)` BEFORE text events
   - `run_react` (react_strategy.py): same pattern in the LLM response handling loop

6. Non-Anthropic warning in `ThinRuntime.run()` (runtime.py):
   - After mode detection and before strategy call: if `effective_config.thinking` is set, resolve provider, if NOT anthropic → yield `RuntimeEvent.status("Warning: extended thinking is only supported for Anthropic models. Provider '{provider}' will ignore thinking configuration.")`
   - Anthropic: no warning, thinking config flows through normally
