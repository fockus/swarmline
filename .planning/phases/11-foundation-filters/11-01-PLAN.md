<task id="2" name="ProjectInstructionFilter implementation" critical="no" depends="1">
<action>
1. Реализовать `ProjectInstructionFilter` в `src/swarmline/project_instruction_filter.py`:

   ```python
   class ProjectInstructionFilter:
       INSTRUCTION_FILES = ["RULES.md", "CLAUDE.md", "AGENTS.md", "GEMINI.md"]

       def __init__(self, cwd: str | Path | None = None, home: str | Path | None = None):
           self._cwd = Path(cwd) if cwd else Path.cwd()
           self._home = Path(home) if home else Path.home()
           self._cache: dict[Path, tuple[float, str]] = {}  # path -> (mtime, content)

       async def filter(self, messages, system_prompt) -> tuple[list[Message], str]:
           instructions = self._collect_instructions()
           if not instructions:
               return messages, system_prompt
           merged = "\n\n".join(instructions)
           return messages, f"{merged}\n\n{system_prompt}" if system_prompt else merged
   ```

2. `_collect_instructions()`:
   - Walk from cwd up to root, then check home
   - At each level: check INSTRUCTION_FILES in priority order, take first found
   - Read content, skip empty files
   - Cache by mtime (stat().st_mtime)
   - Return list ordered: home content first, cwd content last

3. `_read_cached(path: Path) -> str | None`:
   - Check mtime in cache
   - If same → return cached content
   - If different → re-read, update cache
   - If file doesn't exist → return None

4. Security: skip symlinks that resolve outside the walk path (like skills/loader.py pattern)

5. Export из `__init__.py` если нужно
</action>

<task id="4" name="SystemReminderFilter implementation" critical="no" depends="3">
<action>
1. Реализовать `SystemReminderFilter.filter()`:

   ```python
   async def filter(self, messages, system_prompt):
       # 1. Evaluate triggers
       active = [r for r in self._reminders if self._is_active(r, messages, system_prompt)]
       if not active:
           return messages, system_prompt

       # 2. Sort by priority (descending)
       active.sort(key=lambda r: r.priority, reverse=True)

       # 3. Budget enforcement
       selected = []
       used_tokens = 0
       for reminder in active:
           tokens = reminder.token_estimate or self._estimate_tokens(reminder.content)
           if used_tokens + tokens <= self._budget_tokens:
               selected.append(reminder)
               used_tokens += tokens

       if not selected:
           # Budget too small even for highest priority — include it anyway
           selected = [active[0]]

       # 4. Inject as <system-reminder> blocks
       blocks = [f"<system-reminder id=\"{r.id}\">\n{r.content}\n</system-reminder>" for r in selected]
       reminder_text = "\n\n".join(blocks)
       new_prompt = f"{system_prompt}\n\n{reminder_text}" if system_prompt else reminder_text

       return messages, new_prompt
   ```

2. Helper: `_estimate_tokens(text: str) -> int` — `int(len(text) / self.CHARS_PER_TOKEN)`
3. Helper: `_is_active(reminder, messages, system_prompt) -> bool` — вызывает trigger если есть
</action>
