# Plan 12-01: Tool Surface Expansion

Phase: 12
Plan: 01
Status: Ready

## Overview

Add domain allow/block filtering to web_fetch and MCP resource reading to McpClient. Web tools already exist with full provider infrastructure — this plan adds the missing security layer and MCP resource capability.
