<task id="4.1" name="Command interceptor + ThinRuntime integration" critical="no">
<action>
1. Add `command_registry: CommandRegistry | None = None` field to AgentConfig (TYPE_CHECKING import)
2. In `runtime_wiring.py`: pass `command_registry` to `create_kwargs` if not None
3. In `ThinRuntime.__init__`: accept `command_registry` kwarg, store as `self._command_registry`
4. In `ThinRuntime.run()`: after UserPromptSubmit hook (line ~232), before input guardrails:
   - If `self._command_registry` is not None and `self._command_registry.is_command(user_text)`:
     - Parse: `cmd_name, cmd_args = self._command_registry.parse_command(user_text)`
     - Execute: `result = await self._command_registry.execute(cmd_name, cmd_args)`
     - Yield `RuntimeEvent.final(text=result)` with stop reason "command"
     - Fire Stop hook with result text
     - Return (skip LLM entirely)
   - Else: pass through unchanged
5. Write tests FIRST (TDD):
   - test_command_intercepted_skips_llm: /help → direct response, LLM not called
   - test_noncommand_passes_through: regular text → LLM called normally
   - test_no_registry_passes_through: no registry → all text passes through
   - test_unknown_command_returns_error: /unknown → error message from registry
   - test_command_after_hook_transform: hook transforms text → command check uses transformed text
   - test_command_fires_stop_hook: command intercept fires Stop hook
   - test_command_with_args: /topic.new my_goal → correct args passed
   - test_command_registry_wiring_from_agent_config: AgentConfig → ThinRuntime receives registry
</action>
