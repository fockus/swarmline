<task id="1.1.3" critical="yes" depends="1.1.2">
<name>Implement DefaultHookDispatcher (TDD GREEN)</name>
<action>
Add to `src/swarmline/hooks/dispatcher.py`:

`DefaultHookDispatcher(registry: HookRegistry)`:
1. `dispatch_pre_tool(tool_name, tool_input)`:
   - Get hooks from registry: `registry.get_hooks("PreToolUse")`
   - For each hook, check matcher (fnmatch or exact match on tool_name)
   - Call `hook.callback(tool_name=tool_name, tool_input=tool_input)`
   - If callback returns dict with `action: "block"` → return HookResult.block(message)
   - If callback returns dict with `action: "modify"` → return HookResult.modify(modified_input)
   - If callback raises exception → log warning, continue (fail-open)
   - Default: return HookResult.allow()

2. `dispatch_post_tool(tool_name, tool_input, tool_output)`:
   - Get hooks: `registry.get_hooks("PostToolUse")`
   - For each hook, check matcher
   - Call `hook.callback(tool_name=tool_name, tool_output=tool_output)`
   - If returns string → use as new output
   - Return final output or None

3. `dispatch_stop(result_text)`:
   - Get hooks: `registry.get_hooks("Stop")`
   - Call each callback with result_text, catch exceptions

4. `dispatch_user_prompt(prompt)`:
   - Get hooks: `registry.get_hooks("UserPromptSubmit")`
   - Call each callback with prompt
   - If returns string → use as new prompt
   - Return final prompt

Update `src/swarmline/hooks/__init__.py` to export HookDispatcher, HookResult, DefaultHookDispatcher.
</action>
